EasyMatch Pro v3.9

복잡한 데이터 매칭과 정제 작업을 자동화하는 초고속 동기화 솔루션

1. 시스템 개요

본 시스템은 대량의 엑셀 데이터를 특정 조건에 따라 실시간으로 추출하고, 클라우드(GitHub) 및 구글 스프레드시트와의 완벽한 동기화를 통해 기업용 데이터를 중앙 집중 관리하는 데 최적화되어 있습니다.

핵심 특징: Per-Monitor DPI Aware V2 기술 적용으로 고해상도 모니터에서도 선명한 가독성을 제공하며, 4가지 전문가용 테마를 지원합니다.

2. 스마트 매칭 (Smart Match Tab)

단일 파일 또는 현재 열려있는 엑셀 시트를 대상으로 전문적인 필터링과 매칭을 수행합니다.

기능 버튼상세 설명
활성 엑셀 연동현재 띄워져 있는 엑셀 창의 내용을 즉시 분석 소스로 로드합니다.
Peek (헤더 미리보기)파일을 로드하기 전, 선택한 파일의 최상단 컬럼 구조를 미리 확인합니다.
📂 탐색 (Cloud Explorer)GitHub 클라우드에 업로드된 파일을 날짜별로 탐색하여 로컬로 로드합니다.
조건 필터 관리매칭 시 제외하거나 포함할 특정 값들을 체크박스 형태로 선택합니다.
작업 실행설정한 규칙에 따라 데이터를 가공하여 화면에 출력하거나 파일로 저장합니다.

3. 일괄 처리 (Batch Processing Tab)

수백 개의 파일이 담긴 폴더를 통째로 가공하여 하나의 결과물로 병합합니다.

  • 소스 폴더 선택: 원본 Excel/CSV 파일들이 모여있는 폴더를 지정합니다.
  • 결과 합치기 옵션: 각 파일의 결과물을 하나로 묶어 대용량 통합 문서를 생성합니다.
  • 공통 참조 파일: 모든 파일에 공통으로 매칭할 기준 정보(예: 마스터 주소록)를 설정합니다.

4. 데이터 품질 관리 (Data Cleaner Tab)

데이터의 무결성을 높이기 위해 불필요한 요소를 제거하거나 규격을 통일합니다.

주요 도구기능 설명
중복 제거지정한 기준 컬럼값이 중복된 행을 삭제하여 고유 값만 유지합니다.
공백 제거 (Trim)텍스트 데이터의 앞뒤에 포함된 불필요한 빈 칸을 일괄 삭제합니다.
전화번호 정규화특수문자를 제거하고 숫자만 추출하여 데이터 매칭 확률을 극대화합니다.
결측치(NaN) 처리비어있는 셀을 '0' 또는 지정한 기본값으로 한 번에 채웁니다.

5. 통계 및 대시보드 (Data Insights)

로드된 데이터의 특징을 시각적으로 분석합니다.

- 기본 정보: 전체 행/열의 개수 및 결측치 현황을 실시간 파악합니다.

- Top 10 분포: 선택한 컬럼에서 가장 빈도가 높은 10개 항목을 가독성 높은 막대 차트로 가시화합니다.

6. 관리자 및 동기화 설정 (Admin Settings)

프로그램의 핵심 보안 및 연동 로직을 제어합니다. (암호: 3867)

구분설정 세부 사항
License기기 고유 ID에 기반한 라이센스 키를 등록하여 기능을 활성화합니다.
GitHub SyncPersonal Access Token 및 저장소 URL을 입력하여 클라우드 팀 협업 환경을 구축합니다.
Network사내 보안망 환경에서 'SSL 검증 해제' 및 '프록시 서버' 설정을 지원합니다.
Branding사용자 취향에 맞는 4가지 프리미엄 테마를 선택할 수 있습니다.
